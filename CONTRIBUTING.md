# Contributing

Thank you for your interest in contributing.

Although this is currently a personal project, the repository follows professional development practices.

## Development Workflow

- Create a feature branch from `develop`
- Follow Conventional Commits
- Open a Pull Request
- Squash and merge after review

## Commit Convention

Examples:

feat(home): add hero section

fix(nav): resolve mobile overflow

refactor(ui): simplify button variants

perf(images): optimize image loading

## Code Style

- TypeScript
- ESLint
- Prettier
- Strict typing
- Reusable components
