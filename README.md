# Portfolio v3

> A personal portfolio built as a production-grade web application.

---

## Overview

This repository contains the source code for the next iteration of my personal portfolio.

Rather than being a traditional portfolio website, this project is being built with the same engineering standards used for modern product applications.

The primary goals are:

* Showcase engineering projects through detailed case studies
* Demonstrate product thinking and software architecture
* Maintain excellent performance, accessibility, and SEO
* Document technical decisions and development process
* Serve as a long-term platform for writing and experimentation

The project will be developed incrementally using a structured, phase-based workflow.

## Development Workflow

Development follows a feature branch workflow.

```
main
│
develop
│
├── feature/phase-01-foundation
├── feature/phase-02-design-system
├── feature/phase-03-layout
├── feature/phase-04-homepage
├── feature/phase-05-work
├── feature/phase-06-case-studies
├── feature/phase-07-blog
├── feature/phase-08-profile
├── feature/phase-09-seo
├── feature/phase-10-performance
├── feature/phase-11-accessibility
└── feature/phase-12-polish
```

Each phase is developed independently, reviewed, and merged into the development branch before being released.
